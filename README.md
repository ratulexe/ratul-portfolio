## 🌟 Features

- **Premium UI/UX Design**: Clean, modern aesthetics with custom tooltips, glassmorphic effects, and subtle micro-animations.
- **Psychological Design Principles**: Integrates UX psychology (like the Zeigarnik effect scroll bar and isolation effect CTAs) to maximize recruiter engagement.
- **Blazing Fast Performance**: Zero frameworks. Pure Vanilla HTML/CSS/JS means the site loads instantly with flawless Lighthouse scores.
- **Fully Responsive**: Flawless layout on ultra-wide desktop monitors, laptops, tablets, and mobile devices without horizontal scrolling issues.
- **Smooth Scrolling & Reveal**: Custom intersection observer animations gracefully reveal content as you scroll down the page.

## 🛠️ Tech Stack

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom design system using CSS variables, Flexbox, and CSS Grid. 
- **Vanilla JavaScript**: Lightweight DOM manipulation for scroll progress, scroll reveals, and mobile navigation toggling.

## 🚀 Getting Started

This project is completely static, meaning no build tools or package managers are required!

### Run locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/ratulexe/ratul-portfolio.git
   ```
2. Navigate into the folder:
   ```bash
   cd ratul-portfolio
   ```
3. Open `index.html` directly in any modern web browser, or use an extension like VS Code's **Live Server** for hot-reloading while you edit.

## 🌐 Deployment

This site is optimized for seamless deployment on static hosting platforms like Vercel, Netlify, or GitHub Pages. 
Simply connect your repository to the platform and it will deploy instantly—no build command required.
